# SaaS Cabinet Dentaire

Projet SaaS pour la gestion de cabinet dentaire.

## Stack Technique

- Frontend: React + Vite + Tailwind CSS
- Backend: Supabase (Auth, Database, Storage)
- Langage: TypeScript

## Installation

1. Cloner le projet
2. Installer les dépendances : `npm install`
3. Créer un fichier `.env` à partir de `.env.example` et configurer les variables.
4. Lancer le projet : `npm run dev`

## Variables d'environnement

Voir `.env.example` pour la liste des variables nécessaires.
